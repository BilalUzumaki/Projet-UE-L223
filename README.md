# Projet-UE-L223
Cook n' Share est une application web qui permet aux utilisateurs de découvrir, partager et gérer des recettes de cuisine.

Ce site a été développé dans le cadre d’un projet universitaire en développement web.

## Fonctionnalités

Consulter différentes recettes

Ajouter et partager ses propres recettes

Interface simple et facile à utiliser

Affichage des ingrédients et des étapes de préparation

## Technologies utilisées

- HTML
- CSS
- PHP
- MySQL
- JavaScript
- XAMPP
- PHPMyAdmin

## Installation

1. Télécharger ou cloner le projet depuis GitHub :

```
git clone https://github.com/votre-utilisateur/Projet-UE-L223.git
```

2. Placer le dossier du projet dans le dossier htdocs de XAMPP.

3. Ouvrir PHPMyAdmin et créer une nouvelle base de données.

4. Importer le fichier SQL situé dans le dossier database.

5. Démarrer Apache et MySQL dans XAMPP.

6. Ouvrir le projet dans votre navigateur :

http://localhost/Projet-UE-L223

## Objectif du projet

L'objectif de ce projet est de mettre en pratique les compétences acquises en développement web, notamment :

- la création d'un site dynamique avec PHP
- la gestion d'une base de données avec MySQL
- la conception d'une interface web simple et fonctionnelle
